# TECHLYFE - Interactive Technology Blog Website

## Project Description

TECHLYFE is an interactive technology blog website developed using HTML, CSS, and JavaScript. The project focuses on enhancing user experience through interactive web elements and dynamic content updates.

This project was created as part of Project 3: Interactive Web Elements in the DecodeLabs Frontend Development Internship Program.

---

## Project Objective

The objective of this project is to implement JavaScript-based interactivity on a webpage using DOM manipulation, event handling, buttons, toggles, and dynamic content updates.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## Features

### Dark Mode Toggle
- Switch between light mode and dark mode.
- Dynamic theme change using JavaScript.

### Dynamic Hero Content
- Updates hero section content when the user clicks a button.
- Demonstrates DOM manipulation.

### Like Button System
- Users can like articles.
- Like count updates dynamically.

### Newsletter Subscription
- User enters an email address.
- Displays a success message after subscription.

### Interactive Trending Topics
- Hover effects on trending technology topics.
