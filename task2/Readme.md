# TECHLYFE - Responsive Technology Blog Website

## Project Description

TECHLYFE is a responsive technology blog website developed using HTML and CSS. The project focuses on creating a flexible and user-friendly layout that adapts seamlessly across different screen sizes, including desktops, tablets, and mobile devices.

This project was developed as part of Project 2: Responsive Web Layout in the DecodeLabs Frontend Development Internship Program.

---

## Project Objective

The objective of this project is to build a responsive webpage that provides an optimal viewing experience on various devices while maintaining proper spacing, alignment, and usability.

---

## Technologies Used

- HTML5
- CSS3
- CSS Flexbox
- CSS Grid
- CSS Media Queries

---

## Key Features

- Fully responsive webpage design
- Responsive navigation menu
- Flexible Grid and Flexbox layouts
- Mobile-friendly content structure
- Proper spacing and alignment across devices
- Optimized typography and images
- Clean and modern user interface

---

## Responsive Design Implementation

The website has been optimized for:

- Desktop Screens
- Laptops
- Tablets
- Mobile Devices

CSS Media Queries have been used to adjust layouts, navigation, spacing, and content positioning based on screen size.
