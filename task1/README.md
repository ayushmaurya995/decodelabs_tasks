# TECHLYFE - Technology Blog Landing Page

## Project Description

TECHLYFE is a static technology blog landing page developed using HTML and CSS. The website provides a modern layout for showcasing technology-related content such as Artificial Intelligence, Software Development, Hardware, and Cybersecurity.

This project was created as part of Project 1: Static Webpage Design in the DecodeLabs Frontend Development Internship Program.

## Technologies Used

- HTML5
- CSS3

## Features

- Semantic HTML structure
- Navigation bar
- Hero section
- Technology categories section
- Latest articles section
- About section
- Newsletter section
- Footer section
- Clean and modern user interface
